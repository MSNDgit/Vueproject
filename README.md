# frontend

## Project setup
```
yarn install
```

### (교재에서는 npm 을 설치 권장하고 있지만 default로 yarn 설치 - 명령어가 다르다)
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
